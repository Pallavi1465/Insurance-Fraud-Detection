
# Insurance Fraud Detection

## Project Description

This project detects whether an insurance claim is fraudulent or legitimate using machine learning models.
Users can enter claim details through a web interface and the system predicts fraud.

## Technologies Used

* Python
* Machine Learning
* Flask
* HTML / CSS
* GitHub

## Features

* Detect fraudulent insurance claims
* User-friendly web interface
* Result page showing prediction

## How to Run

1. Install requirements
2. Run `main.py`
3. Open the web interface in browser
